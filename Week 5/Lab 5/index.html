Making the web for all audiences

The POUR Acronym for Accessibility
Perceivable All the information on a webpage should be perceivable. This means that the user should be able to conciously recognize what you are trying to display to them.
Operable Any expected interaction between the user and the webpage should be in tact. There should be any expected behaviour from the user that the user is not able to complete.
Understandable All the information on a webpage should be understandable. This means that the user should be able to comprehend the point that the information is trying to get accross.
Robust The content on a webpage must be able to be interpreted by a wide audience. The content should also be able to become dynamic with advancing technologies.
